## Setup

```bash
$ npm install
```

Pour plus de confort sur webstorm :

- Installer les plugins "Nest" et "Prettier".
- Cocher l'exécution d'ESLint et Prettier dans "Actions on Save".

## Exécution

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
